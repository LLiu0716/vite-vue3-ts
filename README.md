# vite-vue 3.0 + ts

#### 介绍
重构一下以前学的项目

#### 运行命令
> npm run dev 开发

> npm run build 生产

> npm run dev-b 本地运行生产环境

#### 使用技术栈

1.  vite
2.  vue 3 , vue-router , vueX
3.  vant

3.  你可以 [https://gitee.com/explore](https://gitee.com/explore) 这个地址来了解 Gitee 上的优秀开源项目

5.  Gitee 官方提供的使用手册 [https://gitee.com/help](https://gitee.com/help)
